Fashion E-commerce

This project is a front-end fashion e-commerce website that allows users to browse and interact with a selection of clothing items. It was built using HTML, CSS, and JavaScript, the application features a clean, responsive design that provides an intuitive shopping experience.

Date Converter

This project is a date converter application that enables users to convert dates from numerical formats to written word formats. It is built in Java, and utilizes a graphical user interface (GUI) for an inituitive user experience. Users can choose a file, and the application will process the dates within. Invalid dates are displayed in the console, while valid dates are instantly shown in the GUI.

Snake Game

This is a Snake Game built using Python with the Pygame library. In this game, the player controls a snake that moves around the screen and eats food to grow longer. The goal is to avoid crashing into the walls or the snake's own body while trying to collect as much food as possible.
